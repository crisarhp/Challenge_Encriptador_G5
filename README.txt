¡Bienvenidos al repositorio del proyecto "Sprint 01: Construcción de un Encriptador de texto"!

Este proyecto tiene como objetivo la creación de un encriptador de texto que te permitirá enviar mensajes secretos de forma segura. Esta aplicación web ha sido desarrollada por Cristian Herrera como parte del primer desafío del programa de Oracle Next Education, con el objetivo de poner en práctica los conocimientos adquiridos durante la formación como Principiante en Programación, y está disponible en la página web [nombre de la página web].

En este repositorio, podrás encontrar todos los archivos y documentos relacionados al desarrollo de la aplicación web.

Para utilizar la aplicación, simplemente visita la página web [nombre de la página web] y sigue las instrucciones proporcionadas. La aplicación es fácil de usar y te permitirá encriptar y desencriptar tus mensajes con la clave que hayas compartido con la otra persona.
